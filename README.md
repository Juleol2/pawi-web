This is a personal web project developed to explore the limits of full-stack web development and real-time cloud integration. The application provides a safety net for pet owners by using QR-coded collars to track and locate lost pets.

Tech Stack
Frontend: HTML5, CSS3, JavaScript (Vanilla).

Backend/Database: Firebase (Realtime Database & Authentication).

APIs: Geolocation API & EmailJS (for automated owner notifications).

Key Features
QR Code Integration: Generates unique QR codes for pet collars.

Automatic Geolocation: When a pet's tag is scanned, the system automatically detects the finder's location.

Instant Notifications: Automatically sends an email to the owner with the exact coordinates where the pet was found.

User Management: Fully functional secure Login and Registration system powered by Firebase.

Project Status & Notes
Language: The current interface is in Spanish.

Development: This is a Work-In-Progress (WIP) prototype created to test technical capabilities in location tracking and cloud synchronization.
